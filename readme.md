# プロジェクト作成
```cmd
//プロジェクト作成
django-admin startproject project

cd project

//アプリケーション作成
python manage.py startapp application

//マイグレーションファイル作成
python manage.py makemigrations

//マイグレーション実行
python manage.py migrate

//superuser作成
python manage.py createsuperuser

```
